NahamCon CTF 2022

April 30, 2022 · 2 min read
Nathan Burns
Threat Hunter

Gossip​

Involved escalating privileges, had to get access to a flag that we did not know the location to.
Command to find all SUID binaries: find / -perm -u=s -type f 2>/dev/null Output: ![[Pasted image 20220429115425.png]] /usr/bin/dialog looks interesting, reading its entry on lolbins give us a way to read any file on the system: https://gtfobins.github.io/gtfobins/dialog/
![[Pasted image 20220429120013.png]] Utilizing this, we can read any file on the system. The trick is I don't know where the flag is and we can't run commands, only read any file. After some time trying to find info in /proc or /tmp I realized the way you connect to this challenge is through ssh, maybe they have some keys?

Reading the content of /root/.ssh/id_rsa ![[Pasted image 20220429121313.png]] Success!!! Now we can connect to this machine using this SSH key and find the flag!

Click Me

Given apk file. Open with JADX to view source code. Interesting functions pop out to me:

![[javaw_L5WnB6T4iE.png]]

Need to somehow get CLICKS to over 99 million.. Thought I could make an autoclicker and let it run but if you look at cookieViewClick we can't get any higher than 13371337.

Open APK using adb to see how it works:

![[Pasted image 20220430013853.png]]

So autoclicker is out of the option, lets try to patch this apk and set the required cookies to a small number.

Decompile apk using apk_tool ![[Pasted image 20220430015624.png]]

Open resulting folder in vscode and get ready to change some code Open the entire directory and navigate to \smali\example\clickme\MainActivity.smali Look for the hex value for that large value we saw earlier 99999999 Edit that hex value to just 9. (or any value, just remember it) Recompile apk using apktool: ![[Pasted image 20220430015843.png]]

IMPORTANT​

You must sign the apk, v1 signing wasn't working for me so I jumped to V3 First create a key file: ![[Pasted image 20220430015940.png]] Now sign the apk using apksigner: ![[Pasted image 20220430020024.png]]

Verify results: ![[Pasted image 20220430020041.png]]

Load APK file back into our ADB VM and click until you get to the number you changed in the smali file ![[Pasted image 20220430020134.png]]

Click GET FLAG to see the flag: ![[Pasted image 20220430020154.png]]
